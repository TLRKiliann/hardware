# HARDWARE

Vous trouverez ici quelques fiches pour mieux comprendre le hardware, appliquées à trois types de machines :

- PC fixe
- Laptop
- Smartphone

Au programme :

- composants
- interfaces
- protocoles
- firmware,
- Trame, paquet, segment TCP/IP
- Interactions entre Kernel - RAM - CPU - Process - PID

Et bonne nouvelle : tout est accessible visuellement via des commandes Linux :+1:

Ce n'est pas un guide exhaustif, mais une base solide pour saisir le rôle des composants. Pour les détails techniques, rien ne vous empêche d'aller plus loin, au cas par cas.

Bonne lecture !
